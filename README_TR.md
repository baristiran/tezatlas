## Hızlı Başlangıç

```bash
./scripts/install.sh
```

Tez projenizde `/tez-baslat` yazın ve AI sizi Faz 0'dan başlatır.

## Skill Graph Yapısı

```
skills/INDEX.md           ← harita
  → skills/moc/           ← MOC navigasyon merkezleri
  → skills/core/          ← temel kurallar (Demir Kurallar, Kaynak Politikası vb.)
  → skills/phases/        ← 8 faz
  → skills/techniques/    ← teknikler
  → skills/templates/     ← dosya şablonları
  → skills/tooling/       ← araçlar
  → templates/            ← disiplin, atıf, üniversite şablonları
```
