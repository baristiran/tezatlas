# Hukuk Disiplin Modülü

Bu modül, hukuk alanında tez yazan öğrenciler için disipline özel kuralları içerir.

## Atıf Sistemi

- **Önerilen:** Chicago Notes-Bibliography (Türkiye hukuk fakülteleri standardı)
- **Alternatif:** OSCOLA (İngiliz hukuku), Bluebook (Amerikan hukuku)
- Mevzuat atıfları kaynakçaya dahil EDİLMEZ (metin içi dipnotta verilir)
- Yargı kararları özel formatta atıf yapılır

## Mevzuat Atıf Formatı

### Kanun
```
6098 sayılı Türk Borçlar Kanunu m. 49/1.
```

### Anayasa
```
T.C. Anayasası m. 35 (Mülkiyet Hakkı).
```

### Yönetmelik / Tebliğ
```
[Tam adı], RG: [Tarih], S: [Sayı].
```

### Uluslararası Antlaşma
```
Avrupa İnsan Hakları Sözleşmesi m. 8 (Özel Hayata Saygı Hakkı).
```

## Yargı Kararı Atıf Formatı

### Anayasa Mahkemesi
```
AYM, E. 2020/1, K. 2020/25, T. 15.06.2020.
```

### Yargıtay
```
Yarg. 4. HD, E. 2019/1234, K. 2019/5678, T. 10.03.2019.
```

### Danıştay
```
Dan. 10. D, E. 2018/456, K. 2018/789, T. 22.11.2018.
```

### AİHM
```
[Başvuran adı] / [Devlet], Başvuru No: [no], [tarih].
```

### AB Adalet Divanı
```
ABAD, C-[no], [Taraf adları], [tarih], ECLI:[numara].
```

## Yazım Kuralları

### Dil ve Üslup
- Hukuki terminoloji tutarlı kullanılmalı (TERMINOLOJI.md zorunlu)
- Pasif yapı tercih edilir: "değerlendirilmektedir", "kabul edilmektedir"
- Normatif ifadeler için: "olmalıdır", "gerekmektedir", "zorunludur"
- Betimsel ifadeler için: "öngörmektedir", "düzenlemektedir", "hüküm altına almaktadır"

### Kavramsal Çerçeve
- Her hukuki kavram ilk kullanımda tanımlanmalı
- Doktrindeki farklı görüşler adil şekilde sunulmalı
- Yazar kendi görüşünü "kanaatimizce" ile belirtir

### Karşılaştırmalı Hukuk Bölümü
- En az 3 ülke/sistem karşılaştırılmalı
- Her ülke için: mevzuat + uygulama + yargı kararları
- Karşılaştırma matrisi (tablo) kullanılabilir
- "Hukuki transplant" tehlikesine dikkat çekilmeli

### De Lege Ferenda (Yasa Önerisi)
- Mevcut hukuki boşluk net tanımlanmalı
- Öneri somut madde taslağı şeklinde olabilir
- Gerekçe detaylı açıklanmalı
- Karşılaştırmalı hukuktan ilham belirtilmeli

## Kaynak Kalite Hiyerarşisi (Hukuka Özel)

1. **Birincil kaynaklar:** Mevzuat metinleri, yargı kararları, resmi belgeler
2. **Monografiler:** Tek yazarlı akademik kitaplar
3. **Şerhler/Kommentarlar:** Madde madde kanun açıklamaları
4. **Hakemli dergi makaleleri:** Ankara Üniversitesi HFM, İÜHFM, Galatasaray HFD vb.
5. **Editörlü kitap bölümleri:** Armağan/festschrift'ler
6. **Tezler:** YÖK Tez Merkezi'nden erişilebilir olanlar
7. **Kurum raporları:** BDDK, SPK, TCMB, AB Komisyonu raporları
8. **Çalıştay/konferans bildirileri**

## Tez Yapısı Önerileri

### Klasik Hukuk Tezi Yapısı
```
GİRİŞ
BÖLÜM 1: Kavramsal Çerçeve ve Teorik Temel
BÖLÜM 2: Mevcut Hukuki Düzenleme / Karşılaştırmalı Hukuk
BÖLÜM 3: Değerlendirme ve Öneriler (de lege ferenda)
SONUÇ
```

### Disiplinlerarası Hukuk Tezi
```
GİRİŞ
BÖLÜM 1: [Diğer disiplinin] Teorik Çerçevesi
BÖLÜM 2: Hukuki Çerçeve ve Karşılaştırmalı Analiz
BÖLÜM 3: Sentez — Hukuki Model Önerisi
SONUÇ
```

## Sık Yapılan Hatalar

- Mevzuat maddesini kaynakçaya eklemek (YANLIŞ — dipnotta verilir)
- Yargı kararını okumadan atıf yapmak (YASAK — tam metin okunmalı)
- Tek bir doktrinel görüşü "genel kabul" gibi sunmak
- Karşılaştırmalı hukukta yüzeysel aktarım (sadece mevzuat çevirisi)
- De lege ferenda bölümünü gerekçesiz bırakmak
- Hukuki terimlerde tutarsızlık (aynı kavrama farklı isimler vermek)
