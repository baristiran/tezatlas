# APA 7 Atıf Rehberi

American Psychological Association, 7. Baskı

## Genel Kurallar

- **Metin içi atıf:** (Yazar, Yıl) veya (Yazar, Yıl, s. XX)
- **Kaynakça:** Tez sonunda alfabetik sırayla listelenir
- Doğrudan alıntılarda sayfa numarası ZORUNLU
- Dolaylı alıntılarda sayfa numarası önerilir
- 40 kelimeden uzun alıntılar blok alıntı olarak girintili yazılır

## Kitap

### Tek Yazarlı
**Metin içi:**
```
(Creswell, 2014, s. 45)
Creswell'e (2014) göre...
```

**Kaynakça:**
```
Creswell, J. W. (2014). Research design: Qualitative, quantitative, and mixed methods approaches (4. bs.). Sage.
```

### İki Yazarlı
**Metin içi:**
```
(Smith ve Jones, 2020, s. 78)
```

**Kaynakça:**
```
Smith, A. B. ve Jones, C. D. (2020). Digital currency design. Cambridge University Press.
```

### Üç ve Üzeri Yazar
**Metin içi (her zaman):**
```
(Allen vd., 2023, s. 34)
```

**Kaynakça (20 yazara kadar hepsi):**
```
Allen, S., Kalemli-Özcan, Ş., Prat, A. ve Shin, H. S. (2023). Central bank digital currencies. Cambridge University Press.
```

### Editörlü Kitap
**Kaynakça:**
```
Bainbridge, S. M. ve Henderson, T. (Ed.). (2019). The Cambridge handbook of shareholder engagement and voting. Cambridge University Press.
```

### Kitap Bölümü
**Kaynakça:**
```
Huber, J. (2020). The case for sovereign money. J. Robertson (Ed.), Monetary reform içinde (s. 112-134). Positive Money.
```

### Çeviri Kitap
**Kaynakça:**
```
Foucault, M. (2013). Güvenlik, toprak, nüfus (F. Taylan, Çev.). İstanbul Bilgi Üniversitesi Yayınları. (Orijinal yayın tarihi 2004)
```

## Dergi Makalesi

### DOI ile
**Kaynakça:**
```
Auer, R. ve Böhme, R. (2020). The technology of retail central bank digital currency. BIS Quarterly Review, Mart, 85-100. https://doi.org/10.xxxx/xxxxx
```

### DOI'siz
**Kaynakça:**
```
Aydın, H. (2023). Dijital Türk Lirası ve hukuki çerçeve. Ankara Üniversitesi Hukuk Fakültesi Dergisi, 72(3), 451-489.
```

## Kurum Raporu

### Yazarı Kurum Olan
**Metin içi:**
```
(Bank for International Settlements [BIS], 2020, s. 12)
```
Sonraki atıflarda:
```
(BIS, 2020, s. 15)
```

**Kaynakça:**
```
Bank for International Settlements. (2020). Central bank digital currencies: Foundational principles and core features (Rapor No. 1). https://www.bis.org/publ/othp33.pdf
```

### Bireysel Yazarlı Kurum Raporu
**Kaynakça:**
```
Auer, R., Cornelli, G. ve Frost, J. (2020). Rise of the central bank digital currencies: Drivers, approaches and technologies (BIS Working Papers No. 880). Bank for International Settlements.
```

## Tez

**Kaynakça:**
```
Yılmaz, A. (2024). Merkez bankası dijital paralarının hukuki niteliği [Yüksek lisans tezi, Ankara Sosyal Bilimler Üniversitesi]. YÖK Tez Merkezi.
```

## Konferans Bildirisi

**Kaynakça:**
```
Smith, J. (2023, Mart). CBDC design principles [Bildiri sunumu]. BIS Innovation Summit, Basel, İsviçre.
```

## Çevrimiçi Kaynak

### Web Sayfası
**Kaynakça:**
```
TCMB. (2024, 15 Ocak). Dijital Türk Lirası araştırma geliştirme projesi. https://www.tcmb.gov.tr/dijital-turk-lirasi
```

### Tarihsiz Web Sayfası
**Kaynakça:**
```
World Bank. (t.y.). Central bank digital currency. https://www.worldbank.org/cbdc
```

## Mevzuat

### APA 7'de mevzuat atıfı
**Metin içi:**
```
Türk Borçlar Kanunu (2012) m. 49/1'e göre...
```

**Kaynakça:**
```
Türk Borçlar Kanunu, Kanun No. 6098 (2012). https://www.mevzuat.gov.tr/...
```

NOT: Bazı Türk üniversiteleri mevzuatı kaynakçaya dahil etmez (danışman tercihine bağlı).

## Özel Durumlar

### Aynı Yazarın Aynı Yıldaki Eserleri
```
(Auer, 2020a, s. 12)
(Auer, 2020b, s. 45)
```

### İkincil Kaynak (Aktaran)
```
Simmel'in belirttiği gibi (aktaran Huber, 2023, s. 67)...
```
Kaynakçaya SADECE okuduğunuz kaynak (Huber) eklenir.
UYARI: İkincil kaynak son çare olmalı. Orijinal kaynağı bulun.

### Doğrudan Alıntı (40 kelimeden kısa)
```
Huber'a (2023) göre, "central bank money is the only form of sovereign money" (s. 45).
```

### Blok Alıntı (40+ kelime)
```
Huber (2023) bu konuda şunları belirtmektedir:

    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor
    incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis
    nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
    (s. 112)
```

### Sayfa Numarası Olmayan Elektronik Kaynak
```
(Auer, 2020, "Teknoloji Seçenekleri" bölümü)

```
