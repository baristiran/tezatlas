# Harvard Atıf Rehberi

Harvard Referencing System

## Genel Kurallar

- **Metin içi atıf:** (Yazar Soyadı Yıl, s. XX)
- **Kaynakça (Reference List):** Tez sonunda alfabetik sırayla
- Doğrudan alıntılarda sayfa numarası ZORUNLU
- Dolaylı alıntılarda sayfa numarası önerilir
- 40 kelimeden uzun alıntılar blok alıntı olarak girintili yazılır
- Harvard sisteminin evrensel bir standardı yoktur — üniversiteler arasında küçük farklılıklar olabilir
- Üniversitenizin kendi Harvard kılavuzunu mutlaka kontrol edin

## Kitap

### Tek Yazarlı
**Metin içi:**
```
(Bodin 1576, s. 45)
Bodin'e (1576, s. 45) göre...
```

**Kaynakça:**
```
Bodin, J. (1576) Les six livres de la République. Paris: Fayard.
```

### İki-Üç Yazarlı
**Metin içi:**
```
(Smith ve Jones 2020, s. 78)
```

**Kaynakça:**
```
Smith, A.B. ve Jones, C.D. (2020) Digital currency design. Cambridge: Cambridge University Press.
```

### Dört ve Üzeri Yazar
**Metin içi:**
```
(Allen vd. 2023, s. 34)
```

**Kaynakça (tüm yazarlar listelenir):**
```
Allen, S., Kalemli-Özcan, Ş., Prat, A. ve Shin, H.S. (2023) Central bank digital currencies. Cambridge: Cambridge University Press.
```

### Editörlü Kitap
**Kaynakça:**
```
Bainbridge, S.M. ve Henderson, T. (ed.) (2019) The Cambridge handbook of shareholder engagement and voting. Cambridge: Cambridge University Press.
```

### Kitap Bölümü
**Metin içi:**
```
(Huber 2020, s. 120)
```

**Kaynakça:**
```
Huber, J. (2020) 'The case for sovereign money', Robertson, J. (ed.) Monetary reform içinde. London: Positive Money, s. 112-134.
```

### Çeviri Kitap
**Kaynakça:**
```
Foucault, M. (2013) Güvenlik, toprak, nüfus. Çev. F. Taylan. İstanbul: İstanbul Bilgi Üniversitesi Yayınları. (Orijinal yayın tarihi: 2004)
```

### Baskı Bilgisi (2. baskı ve sonrası)
**Kaynakça:**
```
Creswell, J.W. (2014) Research design: qualitative, quantitative, and mixed methods approaches. 4. bs. Los Angeles: Sage.
```

## Dergi Makalesi

### Basılı Dergi
**Metin içi:**
```
(Aydın 2023, s. 460)
```

**Kaynakça:**
```
Aydın, H. (2023) 'Dijital Türk Lirası ve hukuki çerçeve', Ankara Üniversitesi Hukuk Fakültesi Dergisi, 72(3), s. 451-489.
```

### DOI ile
**Kaynakça:**
```
Auer, R. ve Böhme, R. (2020) 'The technology of retail central bank digital currency', BIS Quarterly Review, Mart, s. 85-100. doi: 10.xxxx/xxxxx.
```

### Çevrimiçi Dergi (DOI'siz)
**Kaynakça:**
```
Yılmaz, A. (2024) 'CBDC ve mahremiyet', Bilişim Hukuku Dergisi, 5(2), s. 34-56. Erişim adresi: https://example.com [Erişim tarihi: 15 Ocak 2025].
```

## Kurum Raporu

### Yazarı Kurum Olan
**Metin içi:**
```
(Bank for International Settlements 2020, s. 12)
```
Sonraki atıflarda:
```
(BIS 2020, s. 15)
```

**Kaynakça:**
```
Bank for International Settlements (2020) Central bank digital currencies: foundational principles and core features. Rapor No. 1. Basel: BIS.
```

### Bireysel Yazarlı Kurum Raporu
**Kaynakça:**
```
Auer, R., Cornelli, G. ve Frost, J. (2020) Rise of the central bank digital currencies: drivers, approaches and technologies. BIS Working Papers No. 880. Basel: Bank for International Settlements.
```

## Tez

**Kaynakça:**
```
Yılmaz, A. (2024) Merkez bankası dijital paralarının hukuki niteliği. Yüksek Lisans Tezi. Ankara Sosyal Bilimler Üniversitesi.
```

## Konferans Bildirisi

**Kaynakça:**
```
Smith, J. (2023) 'CBDC design principles', BIS Innovation Summit, Basel, İsviçre, 15-16 Mart 2023. Basel: BIS, s. 1-15.
```

## Çevrimiçi Kaynak

### Web Sayfası
**Metin içi:**
```
(TCMB 2024)
```

**Kaynakça:**
```
TCMB (2024) Dijital Türk Lirası araştırma geliştirme projesi. Erişim adresi: https://www.tcmb.gov.tr/dijital-turk-lirasi [Erişim tarihi: 15 Ocak 2025].
```

### Tarihsiz Web Sayfası
**Kaynakça:**
```
World Bank (t.y.) Central bank digital currency. Erişim adresi: https://www.worldbank.org/cbdc [Erişim tarihi: 10 Şubat 2025].
```

## Mevzuat

**Metin içi:**
```
Türk Borçlar Kanunu (2012) m. 49/1'e göre...
```

**Kaynakça:**
```
Türk Borçlar Kanunu, Kanun No. 6098 (2012). Erişim adresi: https://www.mevzuat.gov.tr/... [Erişim tarihi: 10 Şubat 2025].
```

NOT: Bazı üniversiteler mevzuatı kaynakçaya dahil etmez. Danışman tercihini kontrol edin.

## Özel Durumlar

### Aynı Yazarın Aynı Yıldaki Eserleri
**Metin içi:**
```
(Auer 2020a, s. 12)
(Auer 2020b, s. 45)
```

**Kaynakça (kronolojik sıra):**
```
Auer, R. (2020a) 'CBDC design choices', BIS Quarterly Review, Haziran, s. 55-70.
Auer, R. (2020b) 'The technology of retail CBDC', BIS Working Papers No. 880.
```

### İkincil Kaynak (Aktaran)
**Metin içi:**
```
Simmel'in belirttiği gibi (aktaran Huber 2023, s. 67)...
```
Kaynakçaya SADECE okuduğunuz kaynak (Huber) eklenir.
**UYARI:** İkincil kaynak son çare olmalı. Orijinal kaynağı bulun.

### Doğrudan Alıntı (40 kelimeden kısa)
```
Huber'a (2023) göre, "central bank money is the only form of sovereign money" (s. 45).
```

### Blok Alıntı (40+ kelime)
```
Huber (2023, s. 112) bu konuda şunları belirtmektedir:

    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor
    incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis
    nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
```
NOT: Blok alıntıda tırnak işareti kullanılmaz. Sayfa numarası giriş cümlesinde verilir.

### Sayfa Numarası Olmayan Elektronik Kaynak
```
(Auer 2020, 'Teknoloji Seçenekleri' bölümü)
(BIS 2020, para. 3)
```

### Birden Fazla Kaynak Aynı Parantezde
```
(Auer 2020; BIS 2020; Smith ve Jones 2019)
```
Kronolojik veya alfabetik sıra kullanılabilir (üniversite tercihine bağlı).

## APA 7 ile Temel Farklar

| Özellik | Harvard | APA 7 |
|---------|---------|-------|
| Yıl parantezi | Parantez dışında: Yazar (Yıl) | Aynı |
| Yıl sonrası virgül | Virgül YOK | Virgül VAR |
| Makale başlığı | Tek tırnak içinde 'başlık' | Tırnak yok, italik değil |
| Kitap başlığı | İtalik | İtalik |
| Dergi adı | İtalik | İtalik |
| "ve" / "&" | Her zaman "ve" | Türkçe: "ve", İngilizce: "&" |
| Erişim tarihi | Her zaman belirtilir | Sadece değişen içerik için |
| Yayınevi yeri | Belirtilir (London: Sage) | Belirtilmez (Sage) |
| Baskı bilgisi | "4. bs." | "(4. bs.)" parantez içinde |
| DOI formatı | doi: 10.xxx | https://doi.org/10.xxx |

## Chicago Author-Date ile Temel Farklar

| Özellik | Harvard | Chicago Author-Date |
|---------|---------|-------------------|
| Makale başlığı | Tek tırnak 'başlık' | Çift tırnak "başlık" |
| Yayınevi yeri | Şehir: Yayınevi | Şehir: Yayınevi |
| Editör kısaltması | (ed.) | ed. (noktalı) |
| Çoklu sayfa | s. 112-134 | 112-134 (s. olmadan) |
| Erişim tarihi | Her zaman | Gerektiğinde |
