# IEEE Atıf Rehberi

Institute of Electrical and Electronics Engineers Citation Style

## Genel Kurallar

- **Metin içi atıf:** Köşeli parantez içinde numara [1], [2], [3]
- **Kaynakça:** Metin içi GEÇİŞ SIRASINA göre numaralandırılır (alfabetik DEĞİL)
- Bir kaynak ilk geçtiği yerdeki numarasını hep korur
- Birden fazla kaynak: [1], [3], [5] veya [1]-[5]

## Kitap

### Tek Yazarlı
**Metin içi:**
```
Bodin [1] bu konuda...
...ifade edilmiştir [1, s. 45].
```

**Kaynakça:**
```
[1] J. Bodin, Les Six Livres de la République. Paris, Fransa: Fayard, 1576.
```

### Çok Yazarlı
```
[2] S. Allen, Ş. Kalemli-Özcan, A. Prat ve H. S. Shin, Central Bank Digital Currencies. Cambridge, İngiltere: Cambridge Univ. Press, 2023.
```

### Kitap Bölümü
```
[3] J. Huber, "The case for sovereign money," in Monetary Reform, J. Robertson, Ed. London, İngiltere: Positive Money, 2020, s. 112-134.
```

## Dergi Makalesi

```
[4] R. Auer ve R. Böhme, "The technology of retail central bank digital currency," BIS Quart. Rev., s. 85-100, Mar. 2020.
```

DOI varsa:
```
[5] H. Aydın, "Dijital Türk Lirası ve hukuki çerçeve," Ankara Üniv. Hukuk Fak. Derg., c. 72, sy. 3, s. 451-489, 2023, doi: 10.xxxx/xxxxx.
```

## Konferans Bildirisi

```
[6] J. Smith, "CBDC design principles," in Proc. BIS Innovation Summit, Basel, İsviçre, Mar. 2023, s. 1-15.
```

## Kurum Raporu / Teknik Rapor

```
[7] Bank for International Settlements, "Central bank digital currencies: Foundational principles and core features," BIS, Basel, İsviçre, Rep. No. 1, 2020.
```

## Tez

```
[8] A. Yılmaz, "Merkez bankası dijital paralarının hukuki niteliği," Yüksek Lisans tezi, Bilişim Huk. ABD, Ankara Sosyal Bilimler Üniv., Ankara, Türkiye, 2024.
```

## Standart

```
[9] ISO 20022:2013, Financial services — Universal financial industry message scheme. Cenevre, İsviçre: ISO, 2013.
```

## Patent

```
[10] A. B. Smith, "Digital currency transaction system," ABD Patenti 10 123 456, Eki. 15, 2023.
```

## Çevrimiçi Kaynak

```
[11] TCMB, "Dijital Türk Lirası araştırma geliştirme projesi," 2024. [Çevrimiçi]. Erişim: https://www.tcmb.gov.tr/dijital-turk-lirasi. [Erişim tarihi: 15 Oca. 2025].
```

## IEEE Kısaltma Kuralları

- Dergi adları kısaltılır (IEEE kısaltma listesine göre)
- Ay kısaltmaları: Oca., Şub., Mar., Nis., May., Haz., Tem., Ağu., Eyl., Eki., Kas., Ara.
- İngilizce: Jan., Feb., Mar., Apr., May, Jun., Jul., Aug., Sep., Oct., Nov., Dec.
- "c." = cilt (volume), "sy." = sayı (number), "s." = sayfa (page)

## Sıralama Örneği

Metinde:
```
Dijital para sistemleri [1] son yıllarda gelişmiştir. BIS [2] ve IMF [3] bu konuda
öncü çalışmalar yapmıştır. Auer ve Böhme [4] teknoloji seçeneklerini incelemiştir.
Smith ve Jones [5] ise güvenlik sorunlarına odaklanmıştır. Daha önce BIS [2] tarafından
belirlenen ilkeler...
```

Not: BIS kaynağı [2] numarasını her yerde korur, yeniden numaralandırılmaz.
