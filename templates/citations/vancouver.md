# Vancouver At\u0131f Rehberi

International Committee of Medical Journal Editors (ICMJE) \u2014 Numaral\u0131 At\u0131f Sistemi

## Genel Kurallar

- **Metin i\u00e7i at\u0131f:** K\u00f6\u015feli parantez i\u00e7inde numara [1], [2], [3]
- **Kaynak\u00e7a:** Metin i\u00e7i GE\u00c7\u0130\u015e SIRASINA g\u00f6re numaraland\u0131r\u0131l\u0131r (alfabetik DE\u011e\u0130L)
- Bir kaynak ilk ge\u00e7ti\u011fi yerdeki numaras\u0131n\u0131 her yerde korur
- Birden fazla kaynak: [1,3,5] veya ard\u0131\u015f\u0131k ise [1-5]
- Sayfa belirtme: [1, s.45] veya [1](s.45) \u2014 dergi tercihine g\u00f6re de\u011fi\u015fir
- 6\u2019dan fazla yazarl\u0131 eserlerde ilk 6 yazar yaz\u0131l\u0131r, ard\u0131ndan \u201cvd.\u201d eklenir
- Dergi adlar\u0131 Index Medicus/MEDLINE k\u0131saltma standard\u0131na g\u00f6re k\u0131salt\u0131l\u0131r
- Yazarlar\u0131n t\u00fcm soyadlar\u0131 ve adlar\u0131n\u0131n ba\u015f harfleri (initials) verilir, nokta konmaz

## Kitap

### Tek Yazarl\u0131
**Metin i\u00e7i:**
```
Hayran [1] bu konuda...
...belirtilmi\u015ftir [1].
```

**Kaynak\u00e7a:**
```
1. Hayran M. Sa\u011fl\u0131k bilimlerinde ara\u015ft\u0131rma ve istatistik y\u00f6ntemler. 3. bask\u0131. Ankara: Nobel T\u0131p Kitabevleri; 2018.
```

### \u0130ki Yazarl\u0131
**Kaynak\u00e7a:**
```
2. Akda\u011f R, Ayar Y. T\u00fcrkiye sa\u011fl\u0131k sistemi: d\u00f6n\u00fc\u015f\u00fcm ve gelecek. \u0130stanbul: Sa\u011fl\u0131k Yay\u0131nc\u0131l\u0131k; 2021.
```

### \u00dc\u00e7-Alt\u0131 Yazarl\u0131
**Kaynak\u00e7a (t\u00fcm yazarlar listelenir):**
```
3. S\u00fcmb\u00fclo\u011flu K, S\u00fcmb\u00fclo\u011flu V, Akda\u011f R, \u00c7elik Y, Y\u0131lmaz A, Demir F. Biyoistatistik. 15. bask\u0131. Ankara: Hatibo\u011flu Yay\u0131nevi; 2020.
```

### Alt\u0131dan Fazla Yazar
**Kaynak\u00e7a (ilk 6 + vd.):**
```
4. \u00d6zkan S, T\u00fcz\u00fcn H, Demir F, Karaa\u011fao\u011flu E, Aycan Z, Metin B, vd. T\u00fcrkiye\u2019de bula\u015f\u0131c\u0131 olmayan hastal\u0131klar. Ankara: Halk Sa\u011fl\u0131\u011f\u0131 Genel M\u00fcd\u00fcrl\u00fc\u011f\u00fc; 2022.
```

### Edit\u00f6rl\u00fc Kitap
**Kaynak\u00e7a:**
```
5. Y\u0131ld\u0131r\u0131m A, \u015eim\u015fek H, edit\u00f6rler. Sosyal bilimlerde nitel ara\u015ft\u0131rma y\u00f6ntemleri. 12. bask\u0131. Ankara: Se\u00e7kin Yay\u0131nc\u0131l\u0131k; 2021.
```

### Kitap B\u00f6l\u00fcm\u00fc
**Kaynak\u00e7a:**
```
6. \u00d6zt\u00fcrk O. Anksiyete bozukluklar\u0131. \u0130\u00e7inde: K\u00f6ro\u011flu E, edit\u00f6r. Psikiyatri temel kitab\u0131. 3. bask\u0131. Ankara: Hekimler Yay\u0131n Birli\u011fi; 2019. s. 345-398.
```

### \u00c7eviri Kitap
**Kaynak\u00e7a:**
```
7. Guyton AC, Hall JE. T\u0131bbi fizyoloji. \u00c7a\u011flayan Ye\u011fen B, \u00e7eviri edit\u00f6r\u00fc. 13. bask\u0131. \u0130stanbul: Nobel T\u0131p Kitabevleri; 2017.
```

## \u00d6zel Durumlar

### Epub Ahead of Print (Bask\u0131 \u00d6ncesi Elektronik Yay\u0131n)

### Geri \u00c7ekilen Makale (Retracted Article)
**Kaynak\u00e7a:**

UYARI: Geri \u00e7ekilmi\u015f makalelere at\u0131f yap\u0131l\u0131rken okuyucunun durumdan haberdar olmas\u0131 sa\u011flanmal\u0131d\u0131r. M\u00fcmk\u00fcnse geri \u00e7ekilmi\u015f makaleleri kaynak olarak KULLANMAYIN.

### Ayn\u0131 Yazar\u0131n Birden Fazla Eseri
Vancouver\u2019da numara sistemi kullan\u0131ld\u0131\u011f\u0131ndan, ayn\u0131 yazar\u0131n eserleri metinde ge\u00e7i\u015f s\u0131ras\u0131na g\u00f6re ayr\u0131 numara al\u0131r. Alfabetik s\u0131ralama yap\u0131lmaz.
```
...Hayran [1] ilk \u00e7al\u0131\u015fmas\u0131nda... Hayran [8] daha sonra...
```

### \u0130kincil Kaynak (Aktaran)
Vancouver\u2019da ikincil kaynak kullan\u0131m\u0131 \u00f6nerilmez. Orijinal kayna\u011fa mutlaka ula\u015f\u0131lmal\u0131d\u0131r. Zorunlu durumlarda:
```
30. Koch R. Die Aetiologie der Tuberkulose. Berl Klin Wochenschr. 1882;19:221-230. Aktaran: Hayran M. T\u0131p tarihi ve enfeksiyon hastal\u0131klar\u0131. Ankara: Nobel; 2020. s. 45.
```

UYARI: \u0130kincil kaynak kullan\u0131m\u0131 t\u0131bbi/sa\u011fl\u0131k bilimlerinde KES\u0130NL\u0130KLE son \u00e7are olmal\u0131d\u0131r. Birincil kayna\u011fa ula\u015fmak i\u00e7in k\u00fct\u00fcphaneleraras\u0131 \u00f6d\u00fcn\u00e7 verme hizmetlerini kullan\u0131n.

### Ki\u015fisel \u0130leti\u015fim
Ki\u015fisel ileti\u015fimler (e-posta, s\u00f6zl\u00fc) kaynak\u00e7aya dahil edilmez, yaln\u0131zca metin i\u00e7inde parantez ile belirtilir:
```

```
